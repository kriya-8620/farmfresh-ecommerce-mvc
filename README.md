# 🛒 FarmFresh E-commerce Web Application

A full-stack e-commerce web application built using Node.js, Express, MongoDB, and EJS following MVC architecture.

This project was developed as a **final semester BCA project**, enabling farmers to register, sell vegetables, and customers to purchase products.

---

## 🚀 Features

### 👤 Authentication System
- Farmer/User Registration
- Login & Logout
- Session-based Authentication
- Secure Route Protection

---

### 🥦 Product Management
- Add vegetable products
- View product listings
- Buy available vegetables
- Product management system

---

### 🛒 Shopping Features
- Add to Cart
- Remove from Cart
- View Cart Items
- Calculate Total Price

---

### 💳 Order & Payment Simulation
- Fake Payment System
- Order Summary Page
- Shipping Details Form
- "Order Now" functionality

(Note: Payment gateway is simulated for demonstration purposes.)

---
